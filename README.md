# ATHAR — Digital Observatory for Gig Economy Activity

ATHAR is an interactive Streamlit dashboard designed to analyze gig economy activity in Saudi Arabia using indirect digital signals such as:

- Search Interest
- Delivery Orders
- Ride Requests
- Freelance Jobs
- Active Users
- Urban Activity

The system supports:
- Interactive filtering by region and date
- CSV dataset upload
- Data preprocessing and cleaning
- Data quality analysis (before and after cleaning)
- Dynamic visualizations and insights
- Downloadable filtered datasets

## Technologies Used
- Python
- Streamlit
- Pandas
- Plotly

## Project Features
- Gig Activity Index (GAI)
- Trend Analysis
- Regional Comparison
- Signal Contribution Analysis
- Data Quality Monitoring
- Interactive Dashboard Visualizations
